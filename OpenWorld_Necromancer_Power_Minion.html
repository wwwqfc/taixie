概述

  • 直伤仆从唤灵师 是 直伤仆从夺魂者 的入门配置，如果你还未扩展 夺魂者 特性分支，可以利用该方案先行体验游戏内容。

特性 与 技能

特性

  • 幕化移除 ：需要更强的清症能力，可以点出。
  • 堕落狂热 ：需要单挑 BOSS 时使用，以提高生存能力。
  • 疾速渴嗜 ：佩戴匕首时，可以点出增加移速。

配置说明

技能使用

伤害技能：

  • 使用斧技能 恐怖之爪 ，然后切换到匕首进行普攻。
  • 尽可能使用 迅捷蝗群 。
  • 对抗超过3名敌人时，切换进 死亡之幕 ，然后使用 污染枷锁 和 生命爆弹 。这套伤害的单体输出比匕首要低，但可以对多个目标造成伤害。
  • 在切进 死亡之幕 前或退出后，都可以使用 恐怖之爪 提升输出。
  • 如果你不需要控制破蔑视，可以使用 鬼影降临 和 死尸僵化 获得更多伤害。

防御技能：

  • 在面对一大波敌人时，记得让你的仆从吸引火力。通过 闪避 和 拉扯 尽可能避免受到敌人的攻击，因为如果在 死亡之幕 时，受到攻击会消耗生命之力，切回本体时，受伤也会影响 超级学者符文 的被动。
  • 鬼影降临 可用于为敌人施加 目盲 。
  • 如果短时间内受到大量伤害，可以切换到 死亡之幕 并以生命之力作为第二管血条使用。
  • 如果生存依然有压力，可以考虑点出特性 堕落狂热 。
  • 威力法池 ：用来补充威能

说明

续航能力：

  • 可以通过 召唤血魔 或 “鲜血”特性分支 的 吸血光环 、 鲜血输送 获得持续的小额治疗。
  • 大多数情况下，你都可以依靠 唤灵腐化 将自身的症状转移出去，如果需要更多清症能力，可以点出 幕化移除 ，或 佩戴副手匕首使用 致命虫群 转移症状。
  • 使用主手匕首时，可以不断使用 生命虹吸 治疗自己。
  • 如果不小心被打入倒地状态，尝试通过 生命榨取 重新站起来。

控制技能（破蔑视）：

  • 末日哀嚎
  • 冲锋陷阵
  • 厄运降临
  • 黑暗契约
  • 鬼影降临
  • 死尸僵化
