直伤敏捷龙魂使

  暴君之刃 [Fan]，风月 [InS]
  泰协内容维护小组
  2021-03-07
  未经作者授权，禁止转载

概述

  • 直伤敏捷龙魂使 是一套集 输出 和 增益提供 于一身的副本配置，它的主要作用是使用 高层命令 为小队常驻 敏捷 ，并为队伍提供 刺客灵气 以及 断魂之峰 提高直伤输出。
  • 直伤敏捷龙魂使 需要与 急速燃火者 的配置 或 有其他 急速 来源的配置 搭配使用。
  • 直伤敏捷龙魂使 是队伍中破蔑视的主力，碎层挑战建议提前准备小道具破蔑视，获取方式请查看：碎层小道具

特性 与 技能

  • 直伤
  • 治疗

特性

  • 残暴冲力 ，实战中尽量避免使用翻滚闪避，以获得该特性提升暴击率。
  • 灰烬形体 ，碎层挑战模式也可以点出该特性，使得施放 冰刃怒火 时，可以利用该特性叠加 5层 易伤 ，以提高队伍的开场爆发。
  • 锻铁意志 ，队伍中有 直伤猎龙者 时，可选用。

技能

  • 传奇刺客姿态 可以打出更高输出，如果不需要 传奇矮人姿态 提供 稳固 ，可以考虑使用该姿态。
  • 传奇恶魔姿态 的 驱逐附魔 可以移除敌人增益，例如遇到雾锁异变 福祸难料 或 复仇 时，可以考虑替换 传奇矮人姿态 。
  • 传奇半人马姿态 的 慰藉屏护 可以提供格挡保护罩，需要摧毁投射物时可以使用，尤其在应对 火焰之血 异变时非常重要。使用时注意与 燃火者 的反射罩子交替配合使用，以达到全程无伤的效果。

特性

  • 锻铁意志 ，队伍中有 直伤猎龙者 时，可选用。

技能

  • 传奇恶魔姿态 的 驱逐附魔 可以移除敌人增益，例如遇到雾锁异变 福祸难料 或 复仇 时，可以考虑替换 传奇矮人姿态 。
  • 传奇半人马姿态 的 慰藉屏护 可以提供格挡保护罩，需要摧毁投射物时可以使用，尤其在应对 火焰之血 异变时非常重要。使用时注意与 燃火者 的反射罩子交替配合使用，以达到全程无伤的效果。

装备 与 消耗品

  • 普通碎层 或 挑战碎层（野队）
  • 挑战碎层（速刷）
  • 普通碎层（治疗）

配置说明

技能使用

主要增益提供：

  • 敏捷 ：F4 高层命令 冷却即用。
  • 威能 ：F2 英勇指挥 （最高5层卡拉狂热，可提供10层 威能 ）， 超级狼群符文
  • 激怒 ： 超级狼群符文
  • 稳固 ：传奇矮人姿态 稳固激发 （脉冲型）
  • 保护 ：传奇龙魂使姿态下的通用技能均可提供（ 人人为我 ）
  • 抗性 ：传奇恶魔姿态 吸收苦痛
  • 灵魂增益 ：根据切换的姿态提供特定增益。

增益移除：

  • 传奇恶魔姿态 驱逐附魔

格挡投射物：

  • 传奇半人马姿态 慰藉屏护

控制技能（破蔑视）：

  • 法杖5 迷雾激流 ：击退每命中一次可破掉 150点蔑视值，在 破碎观测台 碎层中，可以使用特殊行动技能提前中断该技能施放。
  • 单手剑4 桎梏波动
  • 传奇龙魂使姿态 暗刃勇气
  • 传奇矮人姿态 强制束缚
  • 传奇刺客姿态 翠玉风暴
  • 传奇恶魔姿态 苦痛召唤
  • 传奇半人马姿态 能量爆发
