. 猎龙者 . Dragonhunter

直伤猎龙者

  Roul [SC]、Shetsa [SC]
  泰协内容维护小组
  2021-02-10
  未经作者授权，禁止转载

概述

  • 直伤猎龙者 持续输出能力并不出彩，但其起手爆发潜力非常高，在大型地下城中也有不错的输出。
  • 直伤猎龙者（光辉+正义本能） 不是一套单兵作战的配置，你通常需要有其他 直伤猎龙者 / 急速燃火者 / 龙魂使 队友帮助维持 反弹 的覆盖率，以确保 正义本能 能一直提升暴击率。
  • 这套配置由于搭配了技能 正义之剑 ，因此不适合对抗移动的 BOSS，但在面对有多个转阶段的 BOSS 时，其爆发力表现突出。
  • 直伤猎龙者 可以为目标叠加 易伤 （ 揭露符印 ），并能通过 怨毒纹章 提供不错的破蔑视能力。
  • 直伤猎龙者 可以为团队提供 圣盾 和 稳固 ，点出特性 完美铭刻 时，激活 怨毒纹章 可以将纹章被动效果赋予队友。
  • 该配置的装备较为容易获取，初学者也容易掌握这套配置的输出循环。
  • 该配置不依赖 纪律旗帜 和 眼观六路 ，即可通过 正义本能 和 光耀力量 获得 100%暴击率。

特性

  • 医者惩戒 ，实战中如果不能有效保持 反弹 触发 正义本能 提升暴击率，会选择点出。通常情况下，小队中没有其他 直伤猎龙者 / 急速燃火者 / 龙魂使 施放 愤怒符印 / "坚守阵地！" / 传奇龙魂使姿态 即无法确保 反弹 的覆盖率。实战中，队长应当优先将 直伤猎龙者 / 急速燃火者 / 龙魂使 玩家拉入一个小队。

技能

常规：

  • 尽量在 正义之矛 激活时，使用高输出的技能。此技能与 利刃狂舞 有大致相同的冷却时间，建议两个技能组合施放。
  • 尽可能将 正义之矛 与 愤怒符印 或 惩戒符印 同时使用，来最大化输出。（ 复仇符印 ）
  • 使用 愤怒回旋 时，不要用普通攻击打断技能施放。在施放时，尽量让自身模型与目标模型重叠，以便全部飞刃击中目标，造成最高伤害。
  • 在切换到巨剑后，第二次施放 愤怒回旋 时， 愤怒符印 还在冷却中，你将不得不使用一个没有符印铺垫的 愤怒回旋 。
  • 在对抗大体积 BOSS 时，如果 束缚之刃 射程范围内有一个或多个额外目标，可以移动到BOSS模型的内侧施放，这样会由于多目标穿射而造成更高伤害。

  • 光芒锁链
  • 怨毒纹章 是你最强的破蔑视技能，但应仅在必要时使用，激活它将导致你损失大量的输出。
  • 如有必要，可以使用 正义之矛 和 束缚之刃 的拉拽效果来破除蔑视，但这样做将影响到 巨兽猎杀 的增伤覆盖时间 。
  • 陷阱触发 穿刺之光 以及 巨龙之咽 施加 迟缓 。
