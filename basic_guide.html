蔑视条

  • 蔑视条 是一些怪物的特殊指示条，它会出现在血条的下方。
  • 玩家的控制技能对带有蔑视条的怪物不起直接作用，而是减少蔑视条。
  • 大多数的首领怪物都拥有蔑视条。

蔑视条的三种形态

1、灰色蔑视条

  • 灰色蔑视条表明该怪物不受控制技能影响。
  • 如果怪物的蔑视被清空，也会呈现此形态。

2、蓝绿色蔑视条

  • 大多数首领怪及个别小怪拥有此状态，大多数怪物在拥有此状态时，可以施放特殊技能。
  • 怪物受到移动限制效果 或 硬控制时，会逐步减少蓝绿色蔑视条。
  • 如果控制技能施放的不够连贯，短时间内没有再减少蔑视条，怪物会逐步恢复蔑视条，直到再次施放控制。
  • 一旦蔑视条被清空，怪物就会进入被打破状态，之后转为橙色蔑视条。

3、橙色蔑视条

  • 打破状态结束后，蔑视条会呈现橙色，并逐步增长，此时使用控制技能将无法减少蔑视条，直到蔑视条被填满，变为蓝绿色为止。

蔑视条机制

1、蔑视条被打破后，怪物可能发生其中一种或几种状态：

  • 施法被打断 或 停止当前动作，这是最常见的一种状态。
  • 施加 晕眩 ，使怪物昏迷几秒。
  • 被 击倒 。
  • 施加 暴露 ，使怪物在 5秒内受到的伤害提高 50%。
  • 一些首领怪物会出现不同的状态，具体请查看： 点击前往

2、蔑视条恢复

  • 蔑视条受损后，如果一段时间内不再施加控制，将导致蔑视条再生，此时再使用控制技能即可减缓或阻止恢复。
  • 怪物从打破状态结束后，会出现橙色蔑视条，并逐步增长，此时使用控制技能将无法减少蔑视条，直到蔑视条被填满，变为蓝绿色为止。
  • 个别的怪物，在蔑视条被打破后不会开始恢复，而是在下次使用特殊技能时，才出现新的蓝绿色蔑视条。
  • 极少数的怪物，在蔑视条被打破后不会再出现新蔑视条。

3、动态缩放

  • 伴随着越来越多的玩家参与击杀怪物，会导致怪物的蔑视值出现缩放，以 600点蔑视值的怪物为例，随着玩家数量的增长将呈现以下变化：

4、打断奇效

  • 在对抗部分 BOSS 时，BOSS 达到预定血量可以施放的 特殊技能 或 特殊机制，有时可以在该时刻破掉蔑视，让 BOSS 进入昏迷状态，从而使 特殊技能 或 特殊机制 不会被施放。

症状类控制

  • 症状类的被动效果将无法附加给怪物，转而变为减少蔑视条。
  • 注意症状类控制减少蔑视条会每 0.24秒结算一次，因此，这种破蔑视的方法是将蔑视条逐渐减少。

  • 以下列出症状类控制对蔑视条的具体减少数值。

  • 恐惧 ，100点 / 秒

  • 嘲讽 ，75 点 / 秒
  • 定身 ，50 点 / 秒
  • 迟缓 ，50 点 / 秒
  • 冰冻 ，33 点 / 秒
  • 目盲 ，20 点 / 秒
  • 虚弱 ，20 点 / 秒
  • 残废 ，15 点 / 秒

硬控

  • 硬控会在施加给怪物的瞬间，一次性降低一段蔑视条，减少的具体数值根据以下公式计算：

蔑视条减少值 = 100 × 控制效果持续时间

  • 按此公式计算的硬控效果包括 晕眩 、 昏迷 、 击倒 、 上浮 、 下沉 ，以及技能施加的 冻结（如 深度冻结 ）、变形（如 谦逊纹章 ）、石化（如 石蜥毒液 ）。
  • 控制持续时间不足 1秒的技能，蔑视条减少值固定为 100。超过一秒的非整数持续时间，按照上述公式计算，例如 1.5秒持续时间的技能，蔑视条减少值为 100 × 1.5 = 150。
  • 特性 及 法印 延长控制时间，可以提高蔑视条减少值。
  • 一些没有持续时间的控制技能蔑视值如下：
  • 击退 ：150
  • 拉拽 ：150
  • 击飞 ：232
  • 一些壁障类技能（如 巨龙之咽 ），会根据怪物试图穿过的次数进行结算，具体公式如下：

蔑视条减少值 = 150 × 怪物试图穿过的次数

  • 对于硬控破蔑视的详细说明，请参阅激战2官方wiki： 点击前往

属性

  • 属性 是角色与宠物战斗能力的数值化显示。核心属性有 15种，各属性提供的基本值换算对每个职业都是相同的，但各职业的 基础生命值 不同。
  • 属性 可以通过装备、升级组件、消耗品 以及某些 特性 或 技能 增加。当高级别的角色前往低等级的地图时，属性值 会经过动态缩放。

主属性

  • 主属性 分为四种，两个进攻属性，两个防御属性。
  • 主属性 会根据角色等级提升而增加，角色10级以前每次升级都增加，10级后每逢偶数级增加一次，1级时4个主属性各有 37点，80级时，4个主属性 各有1000点。
  • 威力，提升直接伤害。
  • 精准，增加暴击率。
  • 坚韧，增加护甲值。
  • 体力，增加生命值。

次属性

  • 次属性 不会跟随角色等级而增加数值，并且基础值为 0。
  • 暴击效果，增加暴击伤害。
  • 增益效果，延长增益持续时间。
  • 症状伤害，增加症状造成的伤害。
  • 症状效果，延长症状持续时间。
  • 治疗效果，提高角色治疗量，包括自我治疗。

派生属性

  • 派生属性 根据 主属性 和 次属性 计算得到，派生属性 不能作为装备中的属性组合，但可以通过部分职业特性技能、法印符文 以及 消耗品 直接提高数值。
属性名称 描述 80级基础值 属性获取率
护甲 减少直伤伤害 1000 通过 坚韧 和 装备的 防御 提升 护甲，转化率 1 : 1
生命值 角色的最大生命值 由职业决定 每 1点 体力 提高 10点 生命值
暴击率 直伤攻击或技能触发暴击的几率 5% 每 21点 精准 可提高 1% 暴击率，最高不会超过 100%
暴击伤害 暴击时所能造成的额外伤害 150% 每 15点 暴击效果 可提高 1% 暴击伤害，无上限
症状持续时间 增加所能造成症状的持续时间 0% 每 15点 症状效果 可提高 1% 症状持续时间，最高不会超过 100%
增益持续时间 增加所有应用型增益的持续时间 0% 每 15点 增益效果 可提高 1% 增益持续时间，最高不会超过 100%

职业属性

  • 元素使 ，协调冷却速度，减少 4个协调的冷却时间。
  • 战士 ，爆发技能冷却速度，减少爆发技能的冷却时间。
  • 游侠 ，宠物属性奖励，改善宠物的属性值。
  • 唤灵师 ，生命力池，增加生命之力的容量。
  • 潜行者 ，顺手牵羊冷却速度，减少 顺手牵羊 的冷却时间。
  • 工程师 ，工具带冷却速度，减少工具带技能的冷却时间。
  • 守护者 ，美德冷却速度，减少美德技能的冷却时间。
  • 幻术师 ，粉碎技能冷却速度，减少粉碎技能的冷却时间。
  • 魂武者 ，没有职业属性。

其他属性

  • 武器威力，增加武器技能的直接伤害，只能通过武器获得。
  • 防御，增加 护甲 值，只能通过护甲和盾牌获得。
  • 耐力，角色翻滚闪避需要的能量。
  • 移动速度。
  • 对外治疗效果，提升对他人的治疗量，但不能提升对自身的治疗量。通过一些特性、符文 和 消耗品获得。

特殊属性

  • 这些属性只能在特定的情形下有用。
  • 痛苦抗性，提高对痛苦伤害的减免效果，有碎层专精后，痛苦抗性可以按照比例提供 次属性，仅在迷雾碎层内有效。
  • 金币获取率，增加击败敌人后获得的金币，属于账号加成。
  • 业力获取率，增加完成任务后获得的业力，属于账号加成。
  • 经验寻获率，增加消灭敌人后获取的经验值，属于账号加成。
  • 宝物寻获率，提高找到稀有物品的几率，属于账号加成，也在 英雄：装备 面板中显示。

增益

  • 增益 即一般网游中所说的 buff，它们可以来源于职业技能提供，或是通过触发 法印、符文、消耗品 及 技能组合技 获得，个别技能甚至可以将症状转化为增益。

症状

  • 症状 即一般网游中所说的 debuff 和 dot伤害，它们可以来源于职业技能提供，或是通过触发 法印、符文、消耗品 及 技能组合技 获得，个别技能甚至可以将增益转化为症状。

  • 伤害型症状每秒造成的伤害与施法者的 症状伤害 属性有关，具体请通过顶部的 症状伤害 按钮查看。

  • 无论是 效果堆叠 还是 持续时间堆叠 的 症状，每层都有单独持续时间，满层后再堆叠，会先判断新叠加的持续时间 是否高于持续时间最低的层，如果新叠加的更高，则会将持续时间最低的层移除，转而将新叠加层追加到堆栈末尾，否则新叠加的会被遗弃。

组合技

  • 组合技 (Combo)，是指 组合终结技 作用于 组合技区域 时，可以产生额外的附加效果。这些组合效果在一些特殊情况下，可以发挥巨大的作用。
  • 在 激战2 的职业技能中有 9类 组合技区域 技能，分别是黑暗、幻光、火焰、寒冰、流水、光明、闪电、毒素、烟雾；4类 组合终结技：爆炸，跳跃，物理投射物，旋风。

组合终结技

  • 爆炸：组合效果将作用于爆炸触发区域附近的 敌人 或 盟友，范围360。
  • 跳跃：将组合效果作用于 目标敌人 或 自身。
  • 物理投射物：作用于 目标敌人 或 目标240范围内的盟友。常见的投射物技能会标示出每次弹道投射有20%的几率产生效果，而未标示的则为100%。
  • 旋风：随机向四周发射多个物理投射物，每一束投射物都相当于 物理投射物。

直接伤害

  • 直接伤害，即直伤，指由技能或特性直接造成的伤害，可以暴击，并可被目标的护甲减免伤害。

直伤理论伤害计算

直伤理论伤害值 = 武器威力 × 威力 × 技能伤害系数 / 目标护甲

  • 武器威力：即所佩戴武器的“武器威力”随机分布数。通用技能及精英技能通常不由武器威力影响伤害，而是基于玩家等级来计算。武器威力在技能的持续时间内是固定的，特别是脉冲类技能和引导类技能。
  • 威力：玩家属性面板上的威力值。
  • 技能伤害系数：每项技能都有一个独特的系数来计算造成的伤害。游戏中并不直接显示伤害系数，我们的技能数据库中对伤害系数有额外标注。
  • 目标护甲：目标属性面板上的护甲值。

直伤的最终伤害值会因为以下效果增强或削弱：

  • 暴击会增加直接伤害，暴击的默认伤害为 150% 的基础伤害值，这个百分比可以由暴击效果属性来提高。暴击率最高为 100%。
  • 叠加在自身的 威能 、 激怒 会通过改变属性，提高直伤伤害。
  • 可以通过食物、通用效果、升级组件提高角色属性，从而提高伤害。
  • 一些目标在蔑视条被打破后，可以提高对其造成的直接伤害。
  • 攻击偏斜会导致直接伤害减少 50%。偏斜通常由角色的等级差以及 虚弱 导致。
  • 目标身上的每层 易伤 都可以提高 1% 的直接伤害。
  • 目标身上的 保护 将降低 33% 的直接伤害。

为了稳定的造成直伤伤害，应尽量避免以下情况发生：

  • 自身被 目盲 影响，将导致攻击或技能丢失。
  • 目标被其他单位阻挡或处于闪避状态。
  • 超出攻击或技能范围。
  • 目标被地形或物体阻隔。
  • 目标处于 无敌 状态下。

直伤实际伤害计算

  • 游戏中，直伤的技能通常会标注参考伤害值（由图标 伤害 标识），这个伤害基于以下公式计算得出：

参考伤害值 = 攻击次数 × [ 武器威力平均值 × 威力 × 每一击伤害系数 / 基于目标等级的参照护甲 ]

  • 方括号中的计算结果是经过四舍五入的。
  • 武器威力平均值，是由“武器威力”的 （最大值+最小值 / 2） 得出。通用技能及精英技能通常不由武器威力影响伤害，而是基于玩家等级来计算。
  • 基于目标等级的参照护甲，是由等级决定的，80级时的参考护甲值为 2597。
  • 由于一些技能可能会有多段伤害，每段伤害在实战中，将基于是否命中来计算总伤害量，而非参考的总伤害值。
  • 一些症状为主的技能往往也拥有一个直接伤害。

技能伤害系数

  • 游戏中并未直接列出伤害系数，可以通过以下公式反推算出供参考系数值（仅供80级角色使用，并且由于伤害可能是经过四舍五入后的结果，反推的系数值往往会略有偏差）：

直伤伤害系数 = 2597 × 参考伤害值 / （武器威力 * 威力）

  • 由于一些技能可能会有多段伤害，此时应基于游戏内的提示伤害公式来反推系数值。

症状伤害

  • 症状伤害 是由各种症状造成的持续性伤害，该伤害无法暴击，并且不能被目标的护甲减免伤害。

症状伤害计算

  • 每层症状的 持续时间 和 秒伤 在施加给目标的瞬间即被锁定，症状持续期间改变角色属性面板的 症状持续时间 和 症状伤害 均无法影响这一层的持续时间和伤害。

生命虹吸

  • 生命虹吸 是一种伤害敌人治疗自己的行为，一些技能或特性可以提供该效果。例如： 断魂之峰
  • 生命虹吸 分为两部分：生命虹吸伤害 和 生命虹吸治疗，它们的数值是单独结算的。
  • 生命虹吸伤害 受威力影响，但不会暴击，不受目标护甲减免伤害。
  • 生命虹吸治疗 受治疗效果影响，并且不受 生命虹吸伤害 数值大小影响。

移动速度

  • 移动速度 以 单位 / 秒 进行计量。
  • 侧向移动、后退移动 都要比 向前移动 速度慢许多。
  • 战斗中 向前移动 的速度会降低 30%。
  • 移动速度 会受到增加移动速度状态的影响，有多个状态时，只有最大加成会生效。例如，角色同时受到 空气纹章 和 迅捷 影响，此时只会获得 33% 的移速提升。
