直伤双法杖德鲁伊

  泰协内容维护小组
  2020-04-06
  未经作者授权，禁止转载

概述

  • 直伤双法杖德鲁伊 的配置主要用于各种 事件 及 活动 刷怪时使用。
  • 该配置非常易于上手，但不推荐单兵作战时使用。

特性 与 技能

  • 双法杖
  • 法杖 + 长弓

特性

  • 大地恩泽 ，在对抗野外BOSS时点出，为队友叠加 威能 。
  • 精磨砺斧 ，第二武器组使用 斧 / 钉锤 时点出。

技能

治疗技能（可选）：

  • 巨魔药膏 ：持久型治疗。
  • 回春雕文 ：没有施法时间的快速治疗技能。

宠物：

  • 幼龄蓝花楹 （ 蓝花楹的拥抱 ）：最高伤害的宠物。如果你没有《烈焰征途》资料片，可以考虑使用 幼龄猞猁 、 幼龄丛林山猫 或 幼龄猛虎 。
  • 幼龄棕熊 （ 重整旗鼓 ）：具有一定的防御力，主动技能可以清除症状。
  • 幼龄岩石瞪羚 （ 甩头猛冲 ）：用于破蔑视。如果你没有《烈焰征途》资料片，可以考虑使用 幼龄粉色陆行鸟 或 幼龄电翼龙 。
  • 幼龄钢背兽 或 幼龄雾鳞蜥 ：具有一定伤害能力的坦克型宠物。
  • 幼龄湿地龙蜥 （ 湿地虫群 ）：具有一定范围伤害能力的坦克型宠物。
  • 幼龄狼 （ 恐惧嚎叫 ）：可以使用强力的范围控制技能。

特性

  • 大地恩泽 ，在对抗野外BOSS时点出，为队友叠加 威能 。

食物（可选）：

  • 盘装芝麻粉红酒焖鸡 ，团队型大餐。
  • 薄荷怪异莓棒 ，常规选择。
  • 巧克力怪异莓奶油 ，提高宝物寻获率。
  • 勺装薄荷果甜筒冰淇淋 ，提高治疗量。

配置说明

输出技能：

  • 尽可能朝向远距离的敌人使用 日光射线 ，这样该技能的穿透效果就可以伤害其间的其他敌人。
  • 藤蔓缠绕 和 星界灵火 冷却即用。
  • 如果你装备了长弓，也可以调整站位，利用其普攻的穿透效果攻击敌人。
  • 只在你或附近的盟友需要治疗时，才进入 天界化身 。

防御技能：

  • 在万分紧急的情况下，使用 磐石纹章 。
  • 天界化身 状态下有许多技能可以治疗自己及周围的盟友。
  • 崇高转化 用于摧毁投射物。

说明

续航能力：

  • 天界化身 状态下所有技能都可以用于治疗附近队友，其中 回春潮汐 的治疗量最高。
  • 在 生命种子 上施放 月色引力 用于触发组合技清除症状。
  • 在 崇高转化 上施放 先辈恩泽 用于触发组合技获得一次快速治疗。

控制技能（破蔑视）

  • 自然汇流
  • 藤蔓迸射
  • 藤蔓缠绕
