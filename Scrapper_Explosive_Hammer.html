爆破重锤机械师

  紫天龙
  2021-02-17
  未经作者授权，禁止转载

概述

  • 爆破重锤机械师 是一套制造高伤害同时获得屏障来进行防御，同时辅以目盲以及技能无伤输出的高操作难度特性。

特性 与 技能

特性

技能

治疗技能（可选）：

  • 治疗炮台 ：症状多时可以将 H型试剂 更换为此治疗技能。

通用技能（可选）：

  • 净化机械 ：症状多时可以将 漏油跑鞋 更换为此通用技能。

PvP装备

武器 与 法印

巨锤 - 双手
净化法印
能量法印

项链 与 符文

粉碎者项链
抗性符文

配装说明

  • 粉碎者项链 不可更换为 圣骑士项链 ，否则输出过低影响 冲击专家 获取 屏障 的效率。

配置说明

特性说明

  • 爆破进击 ：本套特性的输出来源， 闪避 会重置此特性，受益于 闪光弹 特性， 爆破进击 会对目标和周围目标造成 目盲 ，同时对血量高于90%的目标造成 晕眩 ，同时 晕眩 会触发 系统震荡 和 检查专家 使你获得 屏障 并使对方受到 易伤 和 虚弱 。
  • 钢片炸药 ：该特性的爆炸特指工程技能下的爆炸物会造成 易伤 ，而非爆炸组合技会造成 易伤 。所以爆炸类技能和爆炸终结技不属于同类属性。
  • 速燃引线 ：爆炸类技能会使你获得 激怒 ，爆炸终结技同上不属于爆炸类技能不会触发该特性。
  • 聚能炸药 ：受益于该特性，攻击受到 易伤 的目标，目标每有一层 易伤 除了原有1%之外，每一层还会额外造成0.5%的伤害，同时 雷霆一击 和受益于 钢片炸药 爆炸类技能均可造成 易伤 （例如 精英炮座 下技能和 电子回旋 ）此特性触发 检查专家 也同样提供 易伤 。
  • 聚合药剂 ：如果因为超低血线身上仍有少量症状导致即将倒地（ 困惑 除外），可以尝试为自身提供增益防止倒地。
  • 工程可以根据战术选择刻意被压低血量后触发 隐藏药瓶 特性，获得 威能 和 激怒 后能造成更高伤害。
  • 防护注入 ：被控制后会立即使你获得保护减少受到的直伤伤害33%。

技能说明

  • 壁垒机械 ：效果期间脉冲屏障周围每多一名盟友都会为你额外提供 屏障 （最多5名），同时承受并减少周围盟友受到的33%的直伤伤害（自身不减少），效果期间会有一个跟随的闪电区域，可以配合 火箭冲锋 。
  • 防御力场 ：会为你和周围盟友提供稳固并格挡投射物。
  • 漏油跑鞋 ：在你脚下生成油层，移动时会持续生成并击倒没有稳固的目标，每个目标只能生效一次，不会多次击倒同一个目标，此技能无视格挡和闪避，不会击倒不移动的目标。
  • 药丸弹射 ：高伤害技能，会在近距离目标内弹射多次，不止造成一次面板伤害。
  • 强酸炸弹 ：高伤害技能且无视格挡，本套特性最主要的输出来源之一，本身特效不明显，可以以意外时间触发 爆破进击 干扰对手并造成伤害。跳起时可以按下放下键（默认~键）即可原地跳跃，取消向后跳跃。
  • 药雾熏蒸 ：本身不可为自身移除症状，可以为盟友移除症状，可以用来快速中程触发 爆破进击 或者快速破除例如守护的 愤怒护盾 限次格挡。
  • 超级试剂 ：本特性唯一的移除症状途径，本身会移除一个症状，配合爆炸组合技 强酸炸弹 可以额外移除一个症状，或者搭配 火箭冲锋 获得光明光环，减少10%受到的症状伤害（与特性 应激护甲 的症状减伤会相加）
  • 精英炮座 ：由于携带了抗性符文，每次20秒打开或关闭此工具包都会使你获得4秒抗性来免疫症状伤害，该技能下的 迫击炮弹 伤害最高。
  • 治疗喷雾 ：需要解除昏迷时建议优先此技能，带有再生和威能，如果需要解晕和奔驰极速优选 奔驰疾速 即可。
  • 投掷H型试剂 ：需要抗压时为自己提供 保护 同时属于治疗的腰带技能会触发 步调一致 特性获得奔驰极速方便走位，或者跑点时需要 奔驰疾速 也可使用。
  • 电子回旋 ：反射投射物并对周围目标造成伤害，且由于是爆炸类技能触发 钢片炸药 特性会造成 易伤 。
  • 火箭冲锋 ：重要的闪避输出手段和组合技手段，可以搭配 雷霆一击 或者 壁垒机械 的闪电区域对小范围的群体目标造成 晕眩 ，同时会触发 系统震荡 特性使你获得 屏障 （屏障等于你 屏障 或 昏迷 目标的数量和次数），该技能的跳跃组合技只有在 初始跳跃 和 最终跳跃 时生效，第二段并不会触发组合技。注意，该技能每次落地起跳期间都不能躲闪攻击，可以被攻击或打断。另外此技能施法中可以切换其它工具包预备使用技能，或者是切换工具包按下放下键（默认~键）打断当前的施法动作。

特性思路

  • 尽可能造成多的输出来获得 屏障 例如 强酸炸弹 ，在对方想使用高伤害技能时及时给予打断，或有两个或以上目标时给予控制，使自己获得 屏障 并打断敌方节奏。
  • 时刻触发闪光弹特性保证敌人关键技能未命中几率，辅以 闪光炮弹 技能的区域性目盲存活。
  • 利用你的 H型试剂 提供的保护，获得治疗的同时减少受到的直伤，减少屏障的损失。
  • 在没有 目盲 和控制时高压力利用 奔驰疾速 和 吸热炮弹 对敌人造成冰冻阻碍追击临时退场。
  • 适时使用 防御力场 防止投掷物并获得稳固防止打断，和 电击护盾 格挡并获得屏障。
  • 本特性原则以少打多为主为队友争取时间，同时单挑不能失败。
  • 备注：本特性虽然解症状少但不存在单挑被症状职业针对的问题，对直伤拥有更高的防御，请注意个人操作，这是一个高难度特性，需要灵活点位走位和适时使用技能。

说明

  • 可以使用大锤武器组，你的精英腰带技能被更换为功能机械。 功能机型 ，在初始的冷却时间内可以在技能范围内生成1个功能机械来终结敌人或者复活队友，该技能在技能范围内每多生成一个功能机械，该技能的冷却时间便会增加50%，同时开启机械（法池）类技能，可以使用各种机械来获得优势。
  • 冲击专家 ：该特性由于翻译错误，正确的翻译应为：当你造成伤害时，15%的伤害将转化为屏障。
  • 应激护甲 ：在拥有屏障时总会使你减少受到的症状伤害20%，在面对症状时应该时刻想办法获得屏障，造成输出触发 冲击专家 获得屏障或者使用带有昏迷和眩晕的技能触发 系统震荡 来获得屏障，同时获得的屏障相比基础值增加15%。
  • 步调一致 ：使用治疗技能时会给予附近己方奔驰极速，使用治疗的腰带技能时会给予自己奔驰极速。
  • 检查专家 ：眩晕或者昏迷目标时会造成一次易伤和虚弱，该特性可以同一时间内触发多个目标，其它效果的控制不会触发该特性（例如 漏油跑鞋 的击倒）。
  • 系统震荡 ：控制敌人时获得屏障，以任何的效果都控制都会触发该特性，包括昏迷、眩晕、击倒、击飞、击退、拉拽、嘲讽、恐惧等。
  • 功能机械 ：受益于 系统震荡 现在生成区域内会造成一次群体眩晕。
